dropwizard-vaadin-meinhotel
==============================

Sourcen zu Sascha Aldas Tutorial zur Entwicklung eines Web-basierten Buchungsportals mit Vaadin  

* mvn install
* java -jar target/dropwizard-vaadin-meinhotel*.jar db migrate config.yml
* java -jar target/dropwizard-vaadin-meinhotel*.jar

oder
* VaadinApplication aus Eclipse heraus starten (zum Aufbau des Datenbankschemas mit den Parametern "db migrate config.yml")

* [http://localhost:8080/vaadin](http://localhost:8080/vaadin) im Browser öffnen
